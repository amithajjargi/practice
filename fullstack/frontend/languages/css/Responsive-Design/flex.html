Flex

Def of Flex:-

The flex CSS shorthand property is the combination of flex-grow, flex-shrink, and flex-basis properties

Importance of flex in CSS

1.It is used to set the length of flexible items.
2.The flex property is much more responsive and mobile-friendly.
3.It is easy to position child elements and the main container.
4.The margin doesn't collapse with the content margins.

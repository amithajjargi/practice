ORDERLIST

Topic Orderlist

def of orderlist:- Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read

  1. used to group a set of related items in a specific order
  2. used to display related information in an easy-to-read and concise way as lists
  3. an ordered list can start with number 1, and continue through 2, 3, 4, and so on
  4. Your ordered list can also start with the letter A and go through B, C, D, and so on
